Can&#8217;t import CSV in Excel with Sophos installed

Aug 30, 2017 • Jonas Hendrickx

When Sophos is installed as an antivirus, you may run into issues when importing *.csv (comma separated values) files in Microsoft Excel.

  1. Enable ‘Legacy Text Import’ via ‘File > Options > Data‘

  2. Check ‘From Text (Legacy)‘. </li>

    • Next, use the legacy wizard which is accessible from the ribbon ‘Data > Get Data > Legacy Wizards > From Text (Legacy)‘ </li> </ol>

      Note: This is a workaround you’ll have to do every time you want to import a *.csv as it’s on a per Excel file setting.
